- `Rtree.py`, `Rtree_neo.py` — Υλοποίηση R-tree δομής
- `WindowQuery.py` — Window range queries
- `NNQuery.py` — Nearest neighbor queries
- `DistanceQuery.py` — Distance range queries
- `Beijing_restaurants.txt`, `rtree.csv` — Δεδομένα εισόδου
- `windowRangeQueries.txt`, `distanceRangeQueries.txt`, `NNQueries.txt` — Queries
- `out_*.txt` — Αρχεία εξόδου
- `Anafora2.pdf` — Αναφορά εργασίας
- `Assignment2.pdf` — Εκφώνηση
